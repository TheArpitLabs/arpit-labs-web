## PROJECT SELECTION CRITERIA

All projects must meet the following criteria:

1. **Verified Repository:** Must be a real, public GitHub repository
2. **Active Development:** Repository must show recent activity
3. **Open Source License:** Must have a recognized open-source license
4. **Documentation:** Must have adequate documentation
5. **Community:** Must have an active community (stars, forks, contributors)
6. **Relevance:** Must be relevant to the engineering domain
7. **Quality:** Must be well-maintained and follow best practices
8. **Attribution:** Original authors and maintainers must be clearly identified
